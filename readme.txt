The easiest way to install Moneypenny Live Chat on your WordPress site!

== Description ==

This plugin makes it easy to install and use Moneypenny LiveChat on your WordPress site. **Please note:** You will need to be a Moneypenny LiveChat client before you can get this plugin to work.

== Installation ==

= Automatic Installation: =

* You can install this plugin directly from your WordPress dashboard's 'Plugins' section. Go to Plugins > Add New and search for Moneypenny LiveChat

= Manual Installation: =

* Unzip contents of this .zip file to your `/wp-content/plugins/` directory.

= After Installation: =

1. Activate the plugin through the 'Plugins' section in WordPress.
2. Go to `MP Live Chat`
3. Check enable. Type in your Moneypenny Widget ID.
4. Enter your GTM code if you wish to track some events.
5. Click "Save Changes"
